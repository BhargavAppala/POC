Guided welcome

The guided welcome is built on top of the angular-intro.js directive, which in turn uses intro.js. Details of the API can be found at the Angular Intro.js github page. The tooltip targets and content are configured in the introOptions object in the controller. Any selectors can be used, but using IDs or unique classes specifically for intro targeting will be the most surgical approach.

Be aware that the relative position of the tooltip is hardcoded in the configuration, and does not adapt dynamically. In cases when the highlighted content is close to full viewport width, the tooltip should be positioned top or bottom.
