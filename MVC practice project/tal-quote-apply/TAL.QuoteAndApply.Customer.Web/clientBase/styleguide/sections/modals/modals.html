Modals

The modals are implemented using ngDialog, and their behavior can be heavily customized though the API of event callback hooks.

Opening the modals

/* The open method */ ngDialog.open(options); /* Example of opening a specific modal, which cannot be dismissed except using standard navigation, or clicking away. */ ngDialog.open({ templateUrl: '/path/to/template/', closeByNavigation: false, closeByEscape: false, closeByDocument: false, });
