Grids

Basis grid

The grid classes use the same naming convention as bootstrap, but include additional breakpoints. The full set of breakpoints is:

  • xxs
  • xs
  • sm
  • md
  • lg
  • xl
  • xxl

Building complex responsive behavior

Using several column classes, you can specify behavior at different breakpoints, like in bootstrap.

Offset classes

Offset classes can be used to insert a space a given number of columns wide, to the left of a column element.
